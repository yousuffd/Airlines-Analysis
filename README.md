# ✈️ Indian Airlines Analysis

A complete exploratory data analysis (EDA) project focused on understanding pricing trends, travel patterns, and operational insights within the Indian domestic airline industry using Python and Jupyter Notebook.

---

## 📌 Project Overview

This project analyzes Indian domestic airline ticket data to uncover meaningful insights related to:

- Flight pricing trends
- Airline performance
- Impact of stops and duration on fare prices
- Booking behavior
- Travel class pricing
- Departure and arrival timing patterns

The analysis was performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn within a Jupyter Notebook environment.

---

# 🎯 Objectives

The primary objectives of this analysis are:

- Understand how airline ticket prices vary
- Identify the cheapest and most expensive airlines
- Analyze the effect of stops on ticket pricing
- Explore how flight duration influences fare cost
- Study pricing differences between Economy and Business class
- Discover trends across major Indian cities
- Generate business insights for travelers and airline operators

---

# 📂 Dataset Information

The dataset contains Indian domestic flight booking information across major airlines and cities.

# 🔍 Key Insights

## ✈️ Airline Pricing
- Air Asia offers some of the lowest Economy fares.
- Air India provides comparatively cheaper Business class options.

## ⏳ Booking Window
- Booking flights 3–7 weeks in advance generally results in lower fares.
- Prices rise significantly closer to departure dates.

## 🛑 Stops vs Price
- Ticket prices increase with the number of stops.
- Non-stop flights are generally more preferred by travelers.

## 🕒 Flight Timing
- Late-night and early-morning flights are usually less expensive.

## 📈 Duration Impact
- Longer flight durations tend to increase ticket prices.

## 🏙️ City Insights
- Delhi often offers lower airfare options.
- Hyderabad appears among the more expensive destinations.
